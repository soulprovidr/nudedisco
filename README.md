## Installation

1. Install [Elixir](https://elixir-lang.org/install.html).
1. Clone the project:

    ```bash
    $ git clone git@github.com:soulprovidr/nudedisco.git
    $ cd nudedisco
    ```

1. Install the project dependencies:

    ```bash
    $ mix deps.get
    ```

1. Run the project locally:

    ```bash
    $ iex -S mix
    ```
